--- layout: null section-type: project title: Project --- ## Project [ManChi](https://hoangmaihuy.github.io/ManChi/) is a Chinese learning web application built on AngularJS 1.x, Bootstrap framework and hosting on GitHub Pages. The application concept is from Chinese dictation test (听写 tīngxiě) which students hear the pronunciation then write Chinese characters. Speaking feature will be updated soon!
